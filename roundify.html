What is Roundify?

Roundify is an application/skin built for Rainmeter that allows the user to change the corner radius of their display. Roundify allows for a high level of customizability and we're just getting started with it. Roundify utilizes Rainmeter to create rounded edges to your screen and/or windows. I decided to use Rainmeter as it seemed like a very versatile program to write in and allowed me to very quickly create something out of nothing. This is however my first time programming a Rainmeter program so there was a lot to learn. Rainmeter does have a very strong and helpful community that helped me along the way and I'll give special thanks to them here.

Features

  • - Corner Radius.
  • - Corner Colour.
  • - Enable Full-screen Corners.
  • - Enable Settings Button.
  • - Padding
  • - Updater
  • - Downdater

Settings

Corner Radius

Description

The corner radius option within the settings menu allows you to change how round the corners will be.

Input Method

This value is scrub friendly so you can scroll to change the value or you can click on it and manually type in the value you want it to be. Just make sure you press enter or the value will not be changed. press ESC at any point of typing to abort and revert to the value before you clicked to change.

Notes

The Corner Radius value cannot be any hire than 100. should you type in a value higher than this it will automatically change to the highest possible value "100".

Corner Colour

Description

The corner Colour option within the settings menu allows you to change colour of the corners surrounding windows and the display.

Input Method

This value relies on a different skin. by clicking the external button, a new window will appear called the Colour Picker+ window. from there you can select what colour you want to use either on the colour spectrum graph or using the eye dropper tool. Press ESC at any time to exit the window and revert to the previous colour.

Notes

You must have the Colour Picker+ skin installed in order for the Setting to work. An install button will appear over the setting if it is not found in the skins directory. Just click the install button and it should install and run as normal.
